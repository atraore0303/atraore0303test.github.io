Student Lockout Kiosk

The student lockout kiosk was designed by Campbell University Engineering students. The purpose of this kiosk is to allow students to still have access to their dorms and other campus facilities in the event of losing or being unable to access their own respective ID cards. We strive to make your campus life as easy as possible.

Receive Student ID

This option will dispense a temporary ID card that will allow students access for 30 Min

Return ID

This option will allow students to return previously dispensed card. Students will not be charged if cards are returned in the 72 hour window.

Rejected ID

This is a secondary return option if your card cannot be properly read when attempting to use the Return ID option.
